# Dataset Versions

This document describes the characteristics and features of different dataset versions.

## hs_dataNew

**Purpose:** Dataset created for PR 9292

**Characteristics:**
- Complete code block extractions
- Web documentation content
- Repository documentation content  
- OpenAPI specification blocks

This version focuses on comprehensive code and documentation extraction to support the requirements of PR 9292.

## hs_data_v2_9292

**Purpose:** Enhanced version of hs_dataNew with additional association data

**Characteristics:**
- All features from hs_dataNew
- Additional association data

This version builds upon hs_dataNew by incorporating association data to provide richer context and relationships between different code elements and documentation.
